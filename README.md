# 📊 Strategic Product Analysis Dashboard

An interactive Tableau dashboard that analyzes product performance, sales trends, profitability, and customer purchasing behavior to help businesses make data-driven strategic decisions.

---

## 📌 Project Overview

The **Strategic Product Analysis Dashboard** provides comprehensive insights into product performance by combining key business metrics into a single interactive visualization.

The dashboard helps answer questions such as:

- Which products generate the highest sales?
- Which categories are the most profitable?
- What products require strategic attention?
- How do different segments contribute to overall revenue?
- Which products should be promoted or discontinued?

---
